# EMPLOYEE_MOOD_TRACKING_PYTHON
# 🧠 Employee Mood Tracking System

A Python-based Employee Mood Tracking System that allows employees to log in, select their current mood, receive motivational content, and save their mood history with the current date and time.

## 📌 Features

- 🔐 Secure Employee Login System
- 😊 Multiple Mood Categories
- 💬 Motivational Quotes
- 🍎 Random Health Tips
- 💡 Random Fun Facts
- 📅 Automatic Date & Time Logging (IST)
- 📁 Mood History saved to CSV
- 👤 Employee-wise Mood Tracking

## 🛠️ Technologies Used

- Python
- Pandas
- Google Colab
- CSV File Handling
- Random Module
- Datetime Module
- ZoneInfo (Asia/Kolkata)

## 📂 Project Files

```
EMPLOYEE_MOOD_PROJECT.ipynb
employees_500_indian_names.csv
MOOD_HISTORY.csv (make a CSV file and upload in Colab)
README.md
```

## 🚀 How to Run

1. Open the project in Google Colab.
2. Upload the employee CSV file.
3. Run all cells (or execute the Python script).
4. Enter your Employee Code and Password.
5. Select your current mood.
6. The application will:
   - Display a motivational quote.
   - Show a random health tip.
   - Show a random fun fact.
   - Save your mood history in `MOOD_HISTORY.csv`.
